# Madotte (纏う)

에이전트 엔지니어링을 신체 역량으로 장착하는 영구 인프라 프로젝트.
본체는 이식 가능한 자산층, 성장은 숙주 프로젝트의 실작업 위에서만.

## 구조

```
madotte/
├── MADOTTE.md    # 헌법 10조 + L0~L3 사다리 (최상위 규범)
├── RULER.md      # 자: 밀봉 과제 T1~T5, 채점표 A/B, 조작적 정의 [봉인 후 불변]
├── MAP.md        # 공역 지도: MiroFish 해부도 + 17챕터 3계층 절단
├── ROADMAP.md    # 7/7 임계 P1~P6 + 6개월 버전표 (v0.1~v1.0, L1/L2/L3 게이트)
├── ALLOC.md      # 모델·컴퓨트 할당표 (헌법 10조 구체화)
├── inbox/        # 최신 소식 유입 밸브 (헌법 7조) — 가설 후보로만 변환
├── skills/       # 승격된 자산 (헌법 4조: 2회 재현 + 숙주 독립성)
├── cycles/       # 사이클 기록 (CYCLE-NN.md, 템플릿 참조)
└── reports/      # 버전 종결 기록 (통과/미통과 무관 의무)
```

## 문서 우선순위

RULER.md (계측) > MADOTTE.md (규범) > ROADMAP.md (계획) > 나머지

## 사용법 (사이클 1회)

1. 숙주 실작업을 밀봉 과제 기준으로 에이전트에 투입 (베이스라인 조건)
2. 개입·붕괴 전량 기록 (RULER §1 정의)
3. 부검 → 변수 1개 수정 → 재실행 → 델타 측정
4. cycles/CYCLE-NN.md로 압축해야 사이클이 닫힘
5. 검증된 절차만 skills/로 승격
