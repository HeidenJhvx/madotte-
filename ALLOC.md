## 0. 내 환경 (기입 완료)

- 로컬 사양: Mac mini M4 / RAM 24GB (통합메모리) / 내장 + 외장하드
- 구독: Claude Pro + GPT Pro
- API: DeepSeek (사용중) · GLM (사용 예정) · Sakana FUGU (예정, 미실행)
- 클라우드: 사용 예정 (예산은 §3 상한 내)
- OpenClaw: 저가/오픈소스 모델 연결 방침 — 현재 DeepSeek 연결중

## 1. 할당 원칙

1. **프론티어 = 빌드타임.** 부검·공역 절단·설계·증류 해부에만.
   구독(Claude Pro/GPT Pro)으로 소화 — API 예산을 여기 쓰지 않는다.
2. **실행 = 최저가 통과 모델.** 채점표 A를 통과하는 가장 싼 모델(기본: DeepSeek)이 정답.
   상위 모델 투입은 "싼 모델이 fail했다"는 기록이 있을 때만 (기록 = cycles/에 1줄).
3. **로컬 vs API**: 반복 대량 호출은 로컬(Ollama, M4 24GB에서 7B~14B 양자화급) 우선 검토.
   판단은 $/1k-태스크 실측치로 — 추정 금지, v0.3~에서 측정 후 이 표를 갱신.
